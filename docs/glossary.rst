########
Glossary
########

.. glossary::


    Julian date (JD)

        The Julian date is a continuous count of days since the beginning of the Julian Period on January 1, 4713 BC.
        

    Phase angle

        The angle between the Sun and the observer as seen from the asteroid. 

    Scattering plane
    
        The plane defined by the Sun, the asteroid, and the observer. 

    SVO

        The Spanish Virtual Observatory (SVO) is a project that aims to provide the astronomical community
        with a set of tools to facilitate the access and analysis of astronomical data.
